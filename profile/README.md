# AltrioPay

A payment gateway built on Solana that settles in USDC. No SOL needed for gas. No bank account required.

## The idea

Inspired by **caffè sospeso** — a tradition from Naples where you pay for two coffees, one for yourself and one left behind for someone in need.

AltrioPay does the same for anything. One payment, split: half for you, half for a stranger. They claim it privately. No identity. No KYC. No paperwork.

**Pay for yourself. Pay for someone else. No one has to know.**
